# Requirements Analysis

The recruitment task describes an industrial company where new technical employees need to learn complex documentation, industry standards, quality procedures, safety rules, specialized tools and certification requirements.

The main business problem is that onboarding takes several months and requires significant involvement from experienced experts.

The proposed solution is an AI assistant that supports self-learning, answers questions based on trusted documentation, generates learning paths and quizzes, and escalates uncertain or safety-critical topics to human experts.
