# Tool Operation Guidelines

Specialized industrial tools must be used according to approved internal instructions.

Before using a tool independently, the employee should understand the tool purpose, required setup, safety limitations, inspection checklist and escalation path for unusual behavior.

Incorrect tool configuration may create safety, quality or production risks.

New employees should perform practical exercises under expert supervision before operating specialized tools independently.

If the tool behavior is unclear, the employee should stop work and ask an experienced engineer.
